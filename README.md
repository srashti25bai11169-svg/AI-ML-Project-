# 📚 Study Time Suggestion AI

## 🧠 Project Overview

The **Study Time Suggestion AI** is a simple Python-based project that helps students decide how many hours they should study daily.

It provides personalized recommendations based on:

* Available free time
* Subject difficulty
* Number of days left before the exam

This project demonstrates basic **Artificial Intelligence concepts using rule-based decision making**.

---

## 🎯 Objectives

* To build a simple AI-based system for study planning
* To help students manage time effectively
* To simulate decision-making similar to AI models

---

## ⚙️ Features

* 📊 Suggests daily study hours
* 🎓 Considers subject difficulty (easy/medium/hard)
* ⏰ Adjusts based on exam urgency
* 🧩 Simple and user-friendly interface
* 💡 Provides helpful study tips

---

## 🛠️ Technologies Used

* Python (No external libraries required)

---

## 🚀 How It Works

The system uses a **rule-based approach**:

1. Assigns study time based on subject difficulty
2. Increases time if exams are near
3. Adjusts according to available free hours
4. Outputs a recommended daily study plan

---

## ▶️ How to Run the Project

1. Install Python (if not already installed)
2. Copy the code into a `.py` file (e.g., `study_ai.py`)
3. Open terminal/command prompt
4. Run the program:

```
python study_ai.py
```

5. Enter the required inputs:

   * Free hours per day
   * Subject difficulty
   * Days left for exam

6. View your personalized study recommendation

---

## 💬 Example Input

```
Free hours: 5  
Difficulty: hard  
Days left: 3  
```

## ✅ Example Output

```
Recommended study time: 5 hours/day  
Tip: Take short breaks to avoid burnout!
```
